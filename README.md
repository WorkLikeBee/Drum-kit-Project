How to Use

You can interact with the drum kit in two ways:

1. Mouse Click

Click on any of the buttons displayed on the screen.

2. Keyboard Keys

Press the following keys:

Key	Sound
w	Tom 1
a	Tom 2
s	Tom 3
d	Tom 4
j	Snare
k	Crash
l	Kick Bass
